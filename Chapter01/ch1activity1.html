The World Wide Web

The World Wide Web, abbreviated as WWW and commonly known as the web, is a system of interlinked hypretext documents accessed via the internet.

With a web browser, one an view the web pages that may contain text, images, videos and other multimedia and navigate between them by using hyperlinks

There are many different types of websites, including :

  • Social Networking
  • Publishing
  • Wikis
  • Shopping and Catalog
  • Search Portal
