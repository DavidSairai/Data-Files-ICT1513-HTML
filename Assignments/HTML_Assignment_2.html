    Chapter 4 - CASCADING STYLE SHEETS

    Cascading Style Sheets or CSS is a language used on websites which is responsible for the look and feel of the website. I learnt that there are three types of ways to use CSS and these are as follows:

    • Internal Style
    • External Style
    • Inline Style

    All the above three styles use style rules to define an HTML element and a set of rules is called a style sheet. Below is a simple use of the three types of CSS styles:

        
        <!--Sample HTML document with the Three Types Of Styles -->
        <!DOCTYPE html>
        <html lang="en">
            <head>
            <title>Use of CSS in HTML</title>
            <!--Internal Style -->
            <style>
            h1 {color:blue;font-family:verdana;border:thin solid;}
            body {font-family:arial;}
            </style>
            <!--Link style -->
            <link href="styles.css" rel="stylesheet">
            <!--Inline Styles-->
            <h1 style="color: blue;">David Sairai</h1>
        <meta http-equiv="Content-Type" content="text/html;charset=UTF-8">
           </head>
        </body>

    A style rule consists of two parts: a selector and a declaration. A selector determines the element to which the rule is applied whilst a declaration is containing with in curly brackets to and details the exact property values

    • Internal Style is only used for that web page ONLY and is created in the <head>section of the document.
    • External Style is used is in an external document and can be linked to multiple webpages to provide uniformity across different properties such type of font, color, size of font, background color, etc. You use the <link> element to be located in the <head> section of the affected document to link the external CSS document to the HTML document.
    • Inline style is used to affect a single element in the document and takes precedence over any other style used on the webpage.

    It is important that the style is applied and referenced in the correct part of the document else you may have inconsistent colors or text fomratting in the website. One must also always write code that properly tabbed and use comments as much as possible to describe each style as much possible to provide information to the next designer what each style is for. Without these comments it wil be difficult to kwno what esch style is and if there are any issues with the code, it wil be difficult to pick up what is causing them.

    Elements in a HTML document are structured in a hierarchal format of parent and child elements where the <html> is the parent element and in the it are child elements such <head> and <body> elements. CSS properties are inherited from parent elements to child elements in what is called inheritance.

    This chapter also covered various selectors which are applied on either elements or techniques such as grouping or a universal selector. These selectors difines the rules to which CSS is applied to elements in a document.
