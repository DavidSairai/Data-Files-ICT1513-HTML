<h1>Web Browser</h1>
A web browser (commonly referred to as a browser) is a software application for retrieving, presenting, and traversing information resources on the World Wide Web. An information resource is identified by a Uniform Resource Identifier (URI) and may be a web page, image, video or other piece of content. Hyperlinks present in resources enable users easily to navigate their browsers to related resources.
<p>Although browsers are primarily intended to use the World Wide Web, they can also be used to access information provided by web servers in private networks or files in file systems.</P>

<h1>Cascading Style Sheets</h1>
<p>Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation semantics (the look and formatting) of a document written in a markup language. Its most common application is to style web pages written in HTML and XHTML, but the language can also be applied to any kind of XML document.</p>
<p>CSS is designed primarily to enable the separation of document content (written in HTML or a similar markup language) from document presentation, including elements such as the layout, colors, and fonts. This separation can improve content accessibility, provide more flexibility and control in the specification of presentation characteristics, enable multiple pages to share formatting, and reduce complexity and repetition in the structural content (such as by allowing for tableless web design). CSS can also allow the same markup page to be presented in different styles for different rendering methods, such as onscreen, in print, by voice (when read out by a speech-based browser or screen reader) and on Braille-based, tactile devices. It can also be used to allow the web page to display differently depending on the screen size or device on which it is being viewed. While the author of a document typically links that document to a CSS file, readers can use a different style sheet, perhaps one on their own computer, to override the one the author has specified.</p>
<p>CSS specifies a priority scheme to determine which style rules apply if more than one rule matches against a particular element. In this so-called cascade, priorities or weights are calculated and assigned to rules, so that the results are predictable.</p>

    Chapter 4 - CASCADING STYLE SHEETS

    Cascading Style Sheets or CSS is a language used on websites which is responsible for the look and feel of the website. I learnt that there are three types of ways to use CSS and these are as follows:

    • Internal Style
    • External Style
    • Inline Style

    All the above three styles use style rules to define an HTML element and a set of rules is called a style sheet. Below is a simple use of the three types of CSS styles:

        
        <!--Sample HTML document with the Three Types Of Styles -->
        <!DOCTYPE html>
        <html lang="en">
            <head>
            <title>Use of CSS in HTML</title>
            <!--Internal Style -->
            <style>
            h1 {color:blue;font-family:verdana;border:thin solid;}
            body {font-family:arial;}
            </style>
            <!--Link style -->
            <link href="styles.css" rel="stylesheet">
            <!--Inline Styles-->
            <h1 style="color: blue;">David Sairai</h1>
        <meta http-equiv="Content-Type" content="text/html;charset=UTF-8">
           </head>
        </body>

    A style rule consists of two parts: a selector and a declaration. A selector determines the element to which the rule is applied whilst a declaration is containing with in curly brackets to and details the exact property values

    • Internal Style is only used for that web page ONLY and is created in the <head>section of the document.
    • External Style is used is in an external document and can be linked to multiple webpages to provide uniformity across different properties such type of font, color, size of font, background color, etc. You use the <link> element to be located in the <head> section of the affected document to link the external CSS document to the HTML document.
    • Inline style is used to affect a single element in the document and takes precedence over any other style used on the webpage.

    One must write code that properly tabbed and use comments to describe each style as much possible to provide information to the next designer what each style is for.

    Elements in a HTML document are structured in a hierarchal format of parent and child elements where the <html> is the parent element and in the it are child elements such <head> and <body> elements. CSS properties are inherited from parent elements to child elements in what is called inheritance.

    This chapter also covered some basic of selection techniques namely

    • Using type selectors â€“ when you apply every element in the document
    • Grouping selectors - when group style using the same rules
    • Combining declarations - when you use trhe same slector with many property declarations
    • Using descendant selectors - when you sue selctor that is descendant of other element
    • Universal Selector - when you apply a style rule to a group of elements
